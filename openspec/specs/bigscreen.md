## 四、四基地地图切换规范

### 4.1 组件结构

四张区域地图叠放于同一位置，切换时通过 JS `setVisible` 控制显隐：

| 基地 | 组件名 | 默认显示 |
|------|--------|----------|
| 浙江常山 | `区域地图_CS` | ✅ 显示 |
| 四川武胜 | `区域地图_WS` | 隐藏 |
| 广西百色 | `区域地图_BS` | 隐藏 |
| 重庆酉阳 | `区域地图_YY` | 隐藏 |

### 4.2 切换按钮逻辑（以切换到武胜为例）

```js
// 1. 切换轮播器
duchamp.getWidgetByName("C9N1").showCardByIndex(1);

// 2. widgetLink 更新 base 参数
// base = "四川武胜"

// 3. JS 地图显隐 + 定位
duchamp.getWidgetByName("区域地图_CS").setVisible(false);
duchamp.getWidgetByName("区域地图_WS").setVisible(true);
duchamp.getWidgetByName("区域地图_BS").setVisible(false);
duchamp.getWidgetByName("区域地图_YY").setVisible(false);
duchamp.getWidgetByName("区域地图_WS").panTo([30.3720862, 106.1657042]);
// panTo 参数顺序：[纬度, 经度]
```

## 五、片区交互规范

### 5.1 点击片区树视图

1. 通过 `FR.remoteEvaluate` 查询 `片区数据-树形` 获取经纬度
2. 调用当前可见地图组件的 `panTo([纬度, 经度])` 定位
3. `block` 参数 `setValue(片区名称)` 触发数据刷新
4. 显示片区信息浮层（10 秒后自动隐藏）

### 5.2 点击地图区域

1. CLICK 事件触发 widgetLink：`block = AREA_NAME`
2. 非空时显示片区信息浮层

### 5.3 Hover Tooltip

通过 `FR.remoteEvaluate` 实时查询，渲染自定义 HTML，包含：片区名称、区域/片区负责人、联系方式、地址、经纬度、种植信息（亩数/株数/大中小树分类）。

## 六、GeoJSON 规范

见独立规范文档：`specs/geojson.md`

核心约束摘要：
- 格式：RFC 7946 FeatureCollection，`-area.json`（MultiPolygon）+ `-point.json`（Point）分离
- 三层层级（FR 地图配置可见）：基地 → 片区 → 地块
- 坐标系：GCJ-02（高德坐标系）
- `properties.name` 与 `ods_ag_base_v2.片区名称` 严格一致（片区级）
- 地块名称参照「农业资产盘点明细.xlsx」
- 部署后须在 FineReport「地图配置」点击「同步地理文件」
